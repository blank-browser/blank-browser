# Blank Browser Vision

## Product

Blank is a quiet, privacy-first browser built on Chromium.

The promise is simple:

```txt
No ads. No noise. No data business.
```

Blank should feel clean and restrained while keeping the real browser
capabilities users expect: modern rendering, extension support, sandboxing,
media support, secure updates, bookmark import, and normal Windows
installation.

## Principles

- Chromium base, not a new engine.
- Private by default, usable by default.
- No ad business.
- No sponsored content.
- No crypto, rewards, wallet, or news feed.
- No invasive telemetry.
- No forced account.
- Optional encrypted sync.
- Security updates treated as mandatory product work.

## MVP Scope

- Chromium-based Windows desktop browser.
- Blank branding and executable identity.
- Minimal new tab inside the browser.
- Built-in ad and tracker blocking.
- Third-party cookie limits where compatibility allows.
- Privacy settings with Standard, Strict, and Burner modes.
- Local bookmarks and browser import.
- Bookmark-only encrypted sync design.
- Windows per-user installer.
- Update strategy documented before public distribution.

## Non-Goals

- No website or landing page in this repository.
- No marketing UI work.
- No fake performance claims.
- No claim of Chrome Sync compatibility.
- No public release without signed binaries and an update plan.
