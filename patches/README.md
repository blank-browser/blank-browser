# Patch Queue

Future Chromium patches live here as docs or generated patch files.

Recommended groups:

```txt
branding/
privacy/
adblock/
sync/
installer/
updater/
```

Rules:

- keep patches small.
- name patches by product area.
- document upstream Chromium revision.
- explain why each patch exists.
- prefer Chromium prefs/flags before source patches.
- test every patch after Chromium security merges.
