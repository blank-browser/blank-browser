# Logo Guidelines

## Current Asset

The repository currently has:

```txt
blanklogo.png
```

Before Chromium branding work starts, convert/export required Windows icon
sizes:

- 16x16
- 24x24
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256

## Rules

- keep app icon simple at small sizes.
- verify against light and dark Windows taskbars.
- verify installer icon.
- verify taskbar grouping.
- verify Alt-Tab icon.
- verify Start Menu icon.

Do not create marketing page assets in this repo.
