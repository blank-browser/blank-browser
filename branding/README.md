# Branding

This folder is for browser product branding assets, not marketing pages.

Current asset in repo root:

```txt
blanklogo.png
```

Generated local asset kit:

```txt
branding/exported/
branding/generated/
```

See:

```txt
docs/assets.md
```

Current assets include:

- app PNGs in common sizes.
- Windows `.ico`.
- installer `.ico`.
- Windows tile PNGs.
- installer wizard BMP/PNG assets.
- light and dark preview exports.

Branding changes must connect to Chromium resource files and installer identity.
