# Scripts

Helper scripts for local Blank Browser development.

Current scripts:

- `check-blank-build.ps1`: verifies local browser binary metadata.
- `run-blank-dev.ps1`: launches the local browser build with an isolated user
  data directory.
- `export-brand-assets.ps1`: exports the basic Blank asset kit from
  `blanklogo.png`.

Scripts should print useful paths, hashes, and errors. They should not hide
destructive operations.
